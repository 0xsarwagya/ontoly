# Code of Conduct

Ontoly is a developer tooling project. We want the community to be direct, technical, and respectful.

## Expected Behavior

- Be precise about bugs and tradeoffs.
- Assume good intent while asking for evidence.
- Keep discussions focused on the project.
- Respect maintainers' time and contributors' boundaries.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks.
- Publishing private information.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may remove comments, close issues, restrict participation, or block accounts when behavior harms the project.

Report conduct issues privately through the repository maintainers.

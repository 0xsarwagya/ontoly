Audit configuration and environment variable usage through Ontoly configuration capabilities. Use when asked where env vars, build config, runtime config, or feature flags are read.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill configuration-analysis
```

## Compatibility

- Skill version: `1.0.0-rc.1`
- Minimum Ontoly version: `1.0.0-rc.1`
- Category: `configuration`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `FindConfiguration`
- `FindConfigurationUsage`
- `FindDependencies`
- `GraphStatistics`
- `EvidencePack`

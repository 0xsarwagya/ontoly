Review repository architecture using Ontoly Software Graph and MCP capabilities. Use when asked to explain architecture, module boundaries, package topology, service ownership, or architectural risk.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill architecture-review
```

## Compatibility

- Skill version: `1.0.0-rc.2`
- Minimum Ontoly version: `1.0.0-rc.2`
- Category: `architecture`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `ExplainArchitecture`
- `GraphStatistics`
- `FindCycles`
- `FindDependencies`
- `EvidencePack`

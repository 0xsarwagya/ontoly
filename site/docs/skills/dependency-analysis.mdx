Analyze internal and package dependencies using Ontoly graph traversal. Use when asked which modules, packages, services, or files depend on each other.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill dependency-analysis
```

## Compatibility

- Skill version: `1.0.0-rc.3`
- Minimum Ontoly version: `1.0.0-rc.3`
- Category: `dependencies`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `FindDependencies`
- `FindDependents`
- `FindCycles`
- `GraphStatistics`
- `EvidencePack`

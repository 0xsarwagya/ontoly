Onboard a developer to a repository using Ontoly graph summaries. Use when asked to explain a new codebase, identify entrypoints, map packages, or suggest first files to inspect.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill codebase-onboarding
```

## Compatibility

- Skill version: `1.0.0-rc.1`
- Minimum Ontoly version: `1.0.0-rc.1`
- Category: `onboarding`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `ExplainArchitecture`
- `FindEntrypoints`
- `GraphStatistics`
- `FindFeatureOwner`
- `EvidencePack`

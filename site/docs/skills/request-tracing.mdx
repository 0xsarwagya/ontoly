Trace request lifecycle behavior through Ontoly route, handler, and call graph capabilities. Use when asked to trace login, API, controller, route, or request execution flow.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill request-tracing
```

## Compatibility

- Skill version: `0.1.0-alpha.3`
- Minimum Ontoly version: `0.1.0-alpha.11`
- Category: `request-flow`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `TraceRequestLifecycle`
- `FindResponsibleFunction`
- `TraceExecution`
- `FindNode`
- `EvidencePack`

Analyze change impact using Ontoly dependency and dependent traversal. Use when asked what breaks if a symbol, service, route, package, or repository node changes or is removed.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill impact-analysis
```

## Compatibility

- Skill version: `0.1.0-alpha.3`
- Minimum Ontoly version: `0.1.0-alpha.6`
- Category: `change-analysis`
- Deprecated: no

## Capabilities

- `ImpactAnalysis`
- `FindDependents`
- `FindDependencies`
- `FindNode`

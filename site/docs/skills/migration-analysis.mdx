Plan migrations using Ontoly architecture, dependency, impact, and configuration evidence. Use when asked to migrate frameworks, packages, APIs, modules, or runtime configuration.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill migration-analysis
```

## Compatibility

- Skill version: `1.0.0-rc.2`
- Minimum Ontoly version: `1.0.0-rc.2`
- Category: `migration`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `ExplainArchitecture`
- `ImpactAnalysis`
- `FindConfigurationUsage`
- `FindDependencies`
- `EvidencePack`

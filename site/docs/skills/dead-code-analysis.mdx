Find potentially unused functions, methods, services, routes, or features using Ontoly dead-code and entrypoint capabilities. Use when asked what code appears unreachable or safe to remove.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill dead-code-analysis
```

## Compatibility

- Skill version: `1.0.0-rc.5`
- Minimum Ontoly version: `1.0.0-rc.5`
- Category: `static-analysis`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `FindDeadCode`
- `FindUnusedFeature`
- `FindEntrypoints`
- `FindDependents`
- `EvidencePack`

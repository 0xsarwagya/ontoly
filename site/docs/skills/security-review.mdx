Review authentication, authorization, and security-sensitive flows using Ontoly graph evidence. Use when asked about auth ownership, protected routes, permissions, guards, or security risk.

## Install

```bash
npx skills add 0xsarwagya/ontoly --skill security-review
```

## Compatibility

- Skill version: `0.1.0-alpha.4`
- Minimum Ontoly version: `0.1.0-alpha.12`
- Category: `security`
- Enhancement: `LLM Enhancement`
- Deprecated: no

## Capabilities

- `FindAuthenticationFlow`
- `FindResponsibleFunction`
- `TraceRequestLifecycle`
- `FindConfigurationUsage`
- `EvidencePack`

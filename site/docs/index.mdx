Ontoly turns source code into a deterministic Software Graph.

It is the semantic layer between a repository and every tool that needs to
understand it: AI agents, MCP servers, SDK generators, documentation systems,
IDEs, architecture visualization, dependency analysis, impact analysis, and
static analysis.

## Core idea

Developer tools should not repeatedly rebuild the same code understanding.
Ontoly builds one graph, validates it, serializes it, and exposes deterministic
queries over it.

## Boundaries

Ontoly does not provide a chat interface, embeddings, vector search, hosted
cloud sync, or AI reasoning. It builds software understanding. Other tools
consume it.

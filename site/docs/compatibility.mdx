Ontoly is TypeScript-native and ESM-first.

Current targets:

- Node.js 22+
- TypeScript 5.9
- pnpm workspaces
- ESM packages

Current parser support:

- `.ts`
- `.tsx`
- `.mts`
- `.cts`

Future parsers should preserve deterministic graph output for identical
input.

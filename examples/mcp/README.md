# MCP Example

Lists deterministic MCP capabilities over the basic example graph.

Run from the repository root.

```sh
pnpm build
pnpm --dir examples/mcp graph
pnpm --dir examples/mcp capabilities
pnpm --dir examples/mcp mcp
pnpm --dir examples/mcp skills
```

The `mcp` command verifies MCP capability discovery. The `skills` command validates the installed-skill workflow and references from the repository root.

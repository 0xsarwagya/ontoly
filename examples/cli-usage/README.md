# CLI Usage Example

Runs common CLI commands against the basic example.

Run from the repository root.

```sh
pnpm build
pnpm --dir examples/cli-usage help
pnpm --dir examples/cli-usage graph
pnpm --dir examples/cli-usage coverage-basic
pnpm --dir examples/cli-usage mcp
pnpm --dir examples/cli-usage skills
```

The `mcp` command verifies MCP capability discovery. The `skills` command validates the installed-skill workflow and references from the repository root.

# TypeScript Library Example

Small library fixture for graph building and symbol inspection.

Run from the repository root.

```sh
pnpm build
pnpm --dir examples/typescript-library graph
pnpm --dir examples/typescript-library inspect
pnpm --dir examples/typescript-library mcp
pnpm --dir examples/typescript-library skills
```

The `mcp` command verifies MCP capability discovery. The `skills` command validates the installed-skill workflow and references from the repository root.

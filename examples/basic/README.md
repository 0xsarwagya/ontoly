# Ontoly Basic Example

Minimal TypeScript project for the first graph build.

Run from the repository root.

```sh
pnpm build
pnpm --dir examples/basic graph
pnpm --dir examples/basic mcp
pnpm --dir examples/basic skills
```

The `mcp` command verifies MCP capability discovery. The `skills` command validates the installed-skill workflow and references from the repository root.

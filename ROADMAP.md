# Ontoly Roadmap

## v0.1

- TypeScript repository discovery.
- Deterministic Software Graph schema.
- TypeScript symbol and relationship extraction.
- Query engine for find, walk, trace, dependencies, callers, callees.
- CLI commands for init, build, inspect, graph, trace, doctor, export, mcp.
- Local graph artifacts under `.ontoly/`.
- Initial MCP-style capabilities.
- Plugin API and Mermaid reference plugin.

## v0.2

- OpenAPI parser.
- SoftwareIR export.
- Graph visualization app.
- More precise TypeScript type resolution.
- RFC-backed graph diffing.

## v0.3

- Prisma parser.
- GraphQL parser.
- Cross-language relationships.
- Incremental invalidation by affected graph region.

## v0.4

- SDK generation plugins.
- Documentation plugins.
- Architecture visualization plugins.
- Benchmark harness across framework repositories.

## v1.0

- Stable Software Graph specification.
- Stable Plugin API.
- Stable Query API.
- Comprehensive documentation.
- Community plugin registry.
